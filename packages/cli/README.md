# Zelaxy CLI

Zelaxy CLI allows you to run [Zelaxy](http://localhost:3000/) using Docker with a single command.

## Installation

To install the Zelaxy CLI globally, use:

```bash
npm install -g zelaxy
```

## Usage

To start Zelaxy, simply run:

```bash
zelaxy
```

### Options

- `-p, --port <port>`: Specify the port to run Zelaxy on (default: 3000).
- `--no-pull`: Skip pulling the latest Docker images.

## Requirements

- Docker must be installed and running on your machine.
