# Tool Integrations

Zelaxy includes 80+ built-in tool integrations that connect your workflows to external services. Tools handle authentication, API calls, and data formatting automatically — you just configure and connect.

## How Tools Work

1. **Add a tool block** to your workflow canvas
2. **Authenticate** via OAuth or API key (one-time setup)
3. **Select an operation** (e.g., "Send Message", "Create Issue")
4. **Configure inputs** using `{{variables}}` from other blocks
5. **Connect to an Agent** for AI-driven tool calling, or use standalone

## Tool as Agent Tool vs Standalone

**Standalone:** The tool block executes its operation directly in the workflow pipeline.

**Agent Tool:** Connect the tool block to an Agent block. The Agent discovers the tool and decides when to call it during reasoning. This enables dynamic tool use — the AI chooses which tools to call based on the conversation.
